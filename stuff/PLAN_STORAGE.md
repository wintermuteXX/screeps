# Plan Storage Guidelines

## Anweisung
Alle erstellten Pläne und TODO-Listen werden automatisch im Ordner `stuff/` gespeichert.

## Format
- Dateinamen: `PLAN_<Beschreibung>.md` oder `TODO_<Datum>_<Beschreibung>.md`
- Format: Markdown
- Inhalt: Strukturierte Pläne mit klaren Aufgaben und Status

## Zweck
- Nachvollziehbarkeit der Änderungen
- Dokumentation der Arbeitsprozesse
- Referenz für zukünftige Arbeiten

## Beispiel-Struktur
\`\`\`markdown
# Plan: [Titel]

## Ziele
- Ziel 1
- Ziel 2

## Aufgaben
- [ ] Aufgabe 1
- [x] Aufgabe 2 (erledigt)

## Notizen
- Wichtige Hinweise
\`\`\`
